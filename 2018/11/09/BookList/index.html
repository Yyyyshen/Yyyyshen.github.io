一份安卓开发书单

2018-11-09 Code Life PV:

辗转了两三次，最后还是走上了安卓开发这条主线。

既然如此，就好好走下去吧，整理了一份经典书单，也算是我的进阶路线，路漫漫其修远兮。

这里记一个Android界的传奇人物——Jake Wharton.人总是要有目标才能前进。

Java

​ 《Java核心技术 卷I》

​ 《Java核心技术 卷II》

​ 《Java编程思想》

​ 《Effective Java》

​ 《深入理解Java虚拟机》

​ 还有一些并发编程的书也需要看。

算法

​ 《算法》

​ 《编程珠玑》

​ 待补充…算法一直是比较头疼的地方，经常有意回避，但终究要面对。

网络

​ 《图解HTTP》

​ 当然还有此系列其他书目，目前只看这本。

操作系统

​ 《Linux就该这么学》

​ 《鸟哥的Linux私房菜》

​ 《深入理解Linux内核》

​ 搞安卓怎么能不懂Linux，不知道有生之年是否走到编系统那一步，似乎很遥远。

安卓

​ （按初级到高级排列）

​ 《第一行代码——Android》

​ 《Android编程权威指南》

​ 《Android群英传》

​ 《Android开发艺术探索》

​ 《C++ Primer》（用NDK写so库需要C++）

​ 原理和架构相关的书不太想整理，因为更安卓新太快，变化大，很多书放到现在都不太合适了，感觉这一块等到基础打好，可以通过看源码来提升，还有优秀的第三方框架也是很好的参考。

​ Gradle也是安卓里一个重点，所以Groovy语法也要懂，不能只会写代码，却不懂编译原理。反编译的话，smali语法又要明白，Android虚拟机还得看。…太多了，慢慢来。

​ 还有不得不说的Kotlin，看势头也快成谷歌亲儿子了，纠结这么多东西还没学完要不要入坑。总之还是要把Java作为重点。

代码

​ 《编写可读代码的艺术》

​ 《设计模式之禅》

​ 《敏捷开发的艺术》

​ 《Git权威指南》

​ 一个好的程序员，不仅要技术上精通，还应当是个艺术家，如何更好的利用工具并且编写出优质的代码也是我们该去追求的。

程序人生

​ 《程序员修炼之道-从小工到专家》

​ 《黑客与画家》

​ 《人间词话》

​ 《浮生六记》

​ 这些就看个人爱好了…

编程之余，看点闲书，弹弹吉他。这就是人生吧。
